# WoW Addon Template

Bas-template för World of Warcraft addon-utveckling med:

- Lua + TOC struktur
- VS Code setup
- Copilot instruktioner
- Stöd för WoWAddonAPIAgents

## Kom igång

1. Skapa ett privat repo i GitHub.
2. Klona repot lokalt.
3. Öppna i VS Code.
4. Installera:
   - GitHub Copilot
   - GitHub Copilot Chat
5. Konfigurera WoWAddonAPIAgents enligt instruktionerna i `docs/copilot-agent-setup.md`

## Struktur

```text
MyAddon/
  MyAddon.toc
  Core.lua
  Modules/
```

## Utveckling

Byt namn på:

- `MyAddon`
- Titel i `.toc`
- SavedVariables

innan du börjar bygga din addon.
